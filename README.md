# VNode
A node system for Unity.

## Keybinds
- **Left Mouse**: Select nodes and ports.  
- **Hold Left Mouse**: Drag nodes or connect ports.  
- **Delete**: Remove selected node(s).  
- **Right Mouse / Context Menu**: Open node context menu.
